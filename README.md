# 🌲 Random Forest Machine Learning Projects

A collection of end-to-end Machine Learning projects demonstrating the implementation of **Random Forest Classification** and **Random Forest Regression** using Scikit-learn. The projects include data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and prediction.

---

## 📂 Projects

### 1. 🏖️ Holiday Package Prediction (Classification)

Predict whether a customer is likely to purchase a holiday package based on demographic and behavioral information.

### Problem Statement
Trips & Travel.Com wants to identify potential customers who are more likely to purchase a newly introduced holiday package. This helps the company improve marketing efficiency and increase conversion rates.

### Features
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Random Forest Classifier
- Model Evaluation

---

### 2. 🚗 Used Car Price Prediction (Regression)

Predict the selling price of a used car using vehicle specifications.

### Problem Statement
Build a regression model capable of estimating the market price of used cars based on important vehicle features.

### Features
- Data Cleaning
- Feature Selection
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Random Forest Regressor
- Model Evaluation

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/random-forest-projects.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

or install manually

```bash
pip install numpy pandas matplotlib seaborn plotly scikit-learn
```

### Run

Open the notebooks in Jupyter Notebook or VS Code.

```
Random Forest Classification Implementation.ipynb
Random Forest Regression Implementation.ipynb
```
